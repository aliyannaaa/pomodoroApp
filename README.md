# 🍅 PomodoroApp

A high-performance mobile Pomodoro Timer built with **Ionic Framework** and **Capacitor**. Designed to help users eliminate distractions and maintain deep focus through timed intervals.

---

## 🚀 Key Features

* **Customizable Intervals:** Set your own work sessions (default 25m), short breaks (5m), and long breaks (15m).
* **Visual Progress:** Circular progress ring and dynamic countdown display.
* **Native Notifications:** Push alerts via Capacitor to notify you when a session ends, even if the app is in the background.
* **Haptic Feedback:** Physical vibration cues when transitioning between "Focus" and "Break" modes.
* **Session History:** Daily stats to track your completed Pomodoro cycles.
* **Cross-Platform:** Single codebase for **iOS**, **Android**, and **Web (PWA)**.

---
